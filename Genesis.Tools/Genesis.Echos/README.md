# Genesis.Echos

ゲーム開発チームの意見を吸い上げて共有するための掲示板。『ゼルダの伝説 ティアーズオブザキングダム』で作られたルピー掲示板のようなものを目指す。

## 機能
- 誰がいつ投稿したかがわかる
- 投稿にタグ付けができる
- 投稿に対して、メンバーアカウントの人たちは「いいね」がつけられる
- 投稿に対して、リーダーアカウントの人たちは「既読」と「コメント」、「意見の重要度（S・A・B）」をつけられる
- 管理者アカウントで、ユーザーのロール変更・削除、投稿の削除ができる

## 技術スタック

## メイン

## 管理者アカウントの設定

`appsettings.json` の `AdminSettings:AdminEmails` に管理者にしたいユーザーのメールアドレスを追加してください。
アプリケーション起動時に、該当ユーザーが存在すれば自動的に管理者ロールに昇格されます。

```json
{
  "AdminSettings": {
    "AdminEmails": [
      "admin@example.com",
      "another-admin@example.com"
    ]
  }
}
```

管理者は以下の機能を使用できます:
- `/admin/users` - ユーザー管理画面
  - ユーザーのロール変更 (Leader ⇄ Member)
  - ユーザーの削除（Adminユーザーは削除不可）
- `/admin/posts` - 投稿管理画面
  - 全投稿の一覧表示
  - 任意の投稿の削除
- 投稿詳細ページで他人の投稿を削除
